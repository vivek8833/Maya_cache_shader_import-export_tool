# post scene file creation Tools - Maya Pipeline Utility

This automation Tools is a PySide6-based pipeline utility for Autodesk Maya.

It automates:
- Shader Export
- Cache Export (Alembic)
- Cache + Shader Build using JSON
- Sequential Maya processing

---

## Requirements
- Autodek Maya
- Python 3.9 (Maya embedded Python)
- PySide6
- Windows OS
